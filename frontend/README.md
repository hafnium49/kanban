# Kanban Frontend

Single-board Kanban app: 5 renamable columns, drag-and-drop cards, add/delete. Client-rendered Next.js, no persistence.

## Stack

Next.js 16 (App Router) - TypeScript - Tailwind CSS v4 - @dnd-kit - Zustand - Vitest - Playwright.

## Scripts

```
npm install
npm run dev       # http://localhost:3000
npm run build
npm run lint
npm test          # vitest unit + component
npm run e2e       # playwright
```
